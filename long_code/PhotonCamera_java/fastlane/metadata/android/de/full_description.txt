<p><i>PhotonCamera</i> verwendet fortschrittliche Algorithmen zum Erfassen und Verarbeiten von Rohbildern, die unübertroffene HDR-Ausgaben liefern. <i>PhotonCamera</i> befindet sich derzeit in der Beta-Phase und entwickelt sich rasant weiter.</p><p><b>Features:</b></p><ul><li>HDRX - Diese Funktion ermöglicht das erweiterte Stapeln vieler unterbelichteter Bilder und erzeugt wunderschöne Ausgaben.</li><li>Verwenden Sie jedes Kameraobjektiv über das Haupt-, Weitwinkel-, Makro-, Tele- oder sogar IR-Objektiv, eine Funktion, die in anderen Open Source-Apps nicht verfügbar ist.</li><li>Manuelle Steuerung - Einfach zu bedienende Regler zur Steuerung von Fokus, Verschlusszeit und ISO unterwegs.<ul><li>Konfigurierbare Einstellungen:</li><li>Anzahl der Frames (maximal)</li><li>Schärfe</li><li>Sättigung</li><li>Schattenstärke</li><li>Interaktiver Sucher</li><li>Aktivieren des Rasters</li><li>Sucherrundung aktivieren</li><li>Erweiterte Moduseinstellungen</li><li>Speichern Sie separate Einstellungen für jedes Kameraobjektiv</li><li>Große Auswahl an unterstützten Geräten.</li></ul></li></ul>
