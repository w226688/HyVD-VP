<p><i>PhotonCamera</i> uses advanced algorithms for capturing and processing raw images which give unmatched HDR outputs. <i>PhotonCamera</i> is currently in beta stage and is having rapid development.</p><p><b>Features:</b></p><ul><li>HDRX - This functionality enables advanced stacking of many underexposed images and creates beautiful outputs.</li><li>Utilize each camera lens, by the main, wide, macro, telephoto or even IR, a functionality not available in other open source apps.</li><li>Manual Control - Easy to use knobs to control Focus, Shutter Speed and ISO on the go.<ul><li>Configurable Settings :</li><li>Number of frames(maximum)</li><li>Sharpness</li><li>Saturation</li><li>Shadow Strength</li><li>Interactive viewfinder</li><li>Enabling the grid</li><li>Enabling viewfinder rounding</li><li>Advanced mode settings</li><li>Save separate settings for each camera lens</li><li>Wide range of supported devices.</li></ul></li></ul>
